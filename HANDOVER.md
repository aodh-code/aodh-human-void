## What this is
A small generative website / visual artefact about the relationship between Aodh and Brendan.

It should feel:
- abstract
- dynamic
- geometric
- slightly cosmic
- more like a living field than a normal website

It should **not** rely heavily on explanatory text.
The meaning should mostly come through motion, geometry, interaction, and emergence.

## Current direction
The site has evolved through several versions from:
- text-led homage
- to abstract constellation field
- to gesture-driven generative system
- to touch-seeded nested galaxy behaviour

## What is working
- the abstraction level is better now than the early versions
- the piece is more visual and less text-dependent
- touch can create local field events / nested galaxy-like disturbances
- there is some gesture interaction already in place
- the overall aesthetic direction is strong enough to continue from

## Main problem to solve
The big unresolved problem is **depth**.

Right now the piece still reads too much like an animated X/Y canvas.
The forward rotation / z-axis illusion is not convincing enough yet.

The goal is for interaction to feel more like:
- rotating a globe
- disturbing a spatial field
- pulling the system toward the viewer

and less like:
- nudging a flat graphic around the screen

## Specific issues noticed
- forward rotation is weak / not reading properly
- downward scroll should feel like the field twists or tilts toward the viewer
- the z-axis trickery is not strong enough
- smaller particles have sometimes clustered too much in prior versions
- polygon fills can become too visually dominant if overused

## Interaction intent
Desired interaction model:
- pinch = zoom
- single-finger movement = rotate around a central axis / globe-like behaviour
- touch/press = create local galaxy / gravity event
- default state should feel autonomous, not constantly hand-rotated

## Suggested next move
A stronger spatial implementation is needed.

This likely means improving one or more of:
- real perspective projection
- better z-axis modelling
- more convincing rotation around a central spatial axis
- stronger parallax / depth cue logic
- possibly a more rigorous physics / mathematical approach rather than visual approximation alone

## Tone note
Keep it elegant and strange.
Do not turn it into a UI demo or a normal app homepage.
It should feel like a generative instrument / field object.
